# I.Software-First-Activity

First activity of Software Engineer

## Assignment

1. **Setup:** Create a repository on GitHub, clone it, and add a collaborator.
2. **Base:** Create `archivo.txt`, run add, commit, and push to `main`.
3. **Branching:** Member A creates `feat-1`, and Member B creates `feat-2`.
4. **Collision:** Both edit the same line of the same file in their branches.
5. **Race:** Member A pushes, opens a PR, Member B approves it, and they merge into `main`.
6. **The Challenge:** Member B tries to merge `feat-2` (GitHub should say it cannot be merged due to conflicts).
7. **Resolution:** Member B must bring the `main` changes into their local branch, resolve the conflict manually, push the solution, and finish the PR.
